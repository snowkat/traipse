# traipse - [dash] as a binary

No really, that's it.

## Usage

```
Usage: traipse [OPTIONS] search-path
  -debug
        show debug messages
  -quiet
        only show errors
```

Provide the path you want to search, and if nothing goes wrong, you'll get a [`Verdict`] from dash as JSON.
